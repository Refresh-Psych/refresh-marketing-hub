Methodology & Data Sources

Primary Data: CMS Open Payments API — General Payment datasets for Program Years 2021-2024. Direct API queries for all 50 states + DC using 3-part specialty filter (Psychiatry) and speaking fee payment type. 955 physicians with verified, aggregated payment data across 51 jurisdictions.

Methodology: Individual payment records queried from CMS DKAN API per state, per year (2021-2024), filtered by Psychiatry specialty and speaking fee payment nature. Records aggregated per physician to produce 4-year totals. Top 50 physicians retained per state (fewer for small states). Company and drug associations derived from payment-linked product data.

Verification: All 956 records are CMS-verified via direct API queries returning individual payment records aggregated per physician. Each entry shows total payments, annual average, sponsoring companies, and promoted drugs.

Limitations: Data covers 2021-2024 only (4 years). Speaking fees prior to 2021 are not included. Some physicians may practice across state lines. Consulting fees are excluded (speaking fees only). VT returned zero psychiatry speaking fee records.
