🔗 Connect Live Analytics Data

Follow these steps to replace sample data with your real GA4 and Google Ads metrics. Total setup time: ~15 minutes.

Step 1: Create the Google Sheet

Easy — 2 minutes
1 Create a new Google Sheet

Go to sheets.google.com and create a new spreadsheet. Name it "Refresh Psychiatry Analytics Data".

2 Set up headers

Rename the first tab to MonthlyData and add these headers in row 1:

Month | Sessions | Organic | Paid | Social | Inquiries | AdSpend | ROAS

(One header per column: A1 = Month, B1 = Sessions, etc.)

Step 2: Add the Automation Script

Medium — 5 minutes
3 Open Apps Script

In your spreadsheet, go to Extensions → Apps Script

4 Enable GA4 API

In the Apps Script editor, click Services (+ icon on the left) → search for "Google Analytics Data API" → click Add.

5 Paste the script

Delete everything in Code.gs and paste the contents of ga4-apps-script.js

6 Add your GA4 Property ID

Find your GA4 Property ID:

  1. Go to analytics.google.com
  2. Click Admin (gear icon)
  3. Under Property, click Property Settings
  4. Copy the Property ID (a number like 123456789)

In the script, replace YOUR_GA4_PROPERTY_ID with your number.

7 Run and authorize

Click Run → select pullGA4MonthlyData. You'll be asked to authorize — click "Advanced" → "Go to Refresh Analytics (unsafe)" → Allow.

This is safe — the script only reads your own GA4 data and writes to your own spreadsheet.

Step 3: Set Up Daily Auto-Run

Easy — 2 minutes
8 Create a trigger

In Apps Script, click the clock icon (Triggers) in the left sidebar → + Add Trigger:

  • Function: dailyFullPull
  • Event source: Time-driven
  • Type: Day timer
  • Time: 5am to 6am

Click Save. The script will now run every morning automatically.

Step 4: Publish and Connect

Easy — 2 minutes
9 Publish the sheet as CSV

Go back to your spreadsheet → File → Share → Publish to web

  • Select the "MonthlyData" sheet
  • Select "Comma-separated values (.csv)" format
  • Click Publish
  • Copy the URL it gives you
10 Give the URL to Claude

Share the published CSV URL and Claude will connect it to your dashboard. The disclaimer will automatically flip from ⚠️ yellow to ✅ green.

Bonus: Google Ads Auto-Export

Optional — 5 minutes

To automatically pull ad spend and ROAS:

  1. Go to ads.google.com → Reports
  2. Create a report with columns: Month, Cost, Impressions, Clicks, Conversions, Conv. value
  3. Click the Schedule button (calendar icon)
  4. Set frequency to Monthly
  5. Destination: Google Sheets → select your analytics spreadsheet
  6. Create a new tab called "GoogleAds"

The mergeGoogleAdsData() function in the script will automatically merge this into your MonthlyData sheet.
